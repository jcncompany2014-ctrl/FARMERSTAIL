## 5) 1:1 통화 가이드 (베타 사용자 7일 후)

### 통화 전 준비
- /admin/cohort 에서 그 사용자 데이터 확인
  - 어떤 박스 받았는지
  - 분석 결과
  - 첫 박스 7일 후 체크인 답변 (있다면)

### 통화 흐름 (30분)

**1) 가벼운 인사 + 강아지 안부 (3분)**
> ○○님 안녕하세요. 시간 내주셔서 감사해요. ○○이는 잘 있죠?

**2) 박스 받은 후 경험 (10분)**
- "박스 받았을 때 첫인상은 어떠셨어요?"
- "○○이가 잘 먹나요? 안 먹는 메뉴 있어요?"
- "변·체중·털 윤기 등 변화 느낀 거 있으세요?"

**3) 서비스 사용 경험 (10분)**
- "분석 받으셨을 때 결과가 도움 됐나요?"
- "앱 / 사이트 사용 중 헷갈리거나 불편한 부분 있었어요?"
- "다른 데서 사료 / 화식 사보신 경험이랑 비교하면 어떠세요?"

**4) 가격 / 정기배송 (5분)**
- "가격은 어떠세요? 부담스럽거나 적당하거나?"
- "정기배송 사용 의향 있으세요? 어떤 조건이면 가입할 거 같아요?"

**5) 마무리 (2분)**
> "솔직한 피드백 정말 도움 됐어요. ○○이를 위해 더 좋은 거 만들어볼게요. 추가로 떠오르는 거 있으면 언제든 메일 주세요."

### 통화 후 필수
1. /admin/cohort 에 메모 저장 (3줄 요약)
2. **즉시 개선 가능한 1가지** 결정 → 그 주에 처리
3. 다음 베타 사용자 인터뷰에서 같은 질문 → 패턴 찾기
