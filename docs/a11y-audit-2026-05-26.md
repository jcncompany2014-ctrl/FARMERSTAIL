## ⚠️ 검토 필요 (R45+ 후속)

### 1. 폼 label 연결 — `htmlFor`/wrap 일관성

현재 `<label>` 이 `<input>` 직접 자식이 아닌 sibling 인 경우가 있어 screen
reader 가 label 을 input 에 연결 못 함. 예:

```tsx
// 현재 (label 과 input 분리)
<label className="..">이름 *</label>
<input type="text" />

// 권장
<label className="..">
  <span>이름 *</span>
  <input type="text" />
</label>
// 또는 htmlFor
<label htmlFor="dog-name">이름 *</label>
<input id="dog-name" type="text" />
```

영향 페이지:
- `/dogs/new` NewDogClient.tsx
- `/dogs/[id]/edit` (EditDogClient 추정)
- 설문 페이지 step 컴포넌트 (다수)

조치: 다음 audit round 에서 일괄 fix.
