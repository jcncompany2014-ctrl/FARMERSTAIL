## 1. Vercel 환경변수 2개 교체 (test_ → live_)
Vercel → farmerstail → Settings → Environment Variables → Production
- `NEXT_PUBLIC_TOSS_CLIENT_KEY` = `live_ck_...` (Toss 개발자센터 운영 클라이언트키)
- `TOSS_SECRET_KEY` = `live_sk_...` (Toss 운영 시크릿키)
- 저장 후 **반드시 재배포** (Deployments → 최신 → Redeploy). NEXT_PUBLIC_ 은 빌드 시점에 박혀서, 재배포 안 하면 옛 키 그대로.

## 2. Toss 콘솔 설정 (등록할 주소는 이미 다 살아있음 — 확인됨 2026-06-11)
- **정기결제(빌링) 사용 신청** ← 자동결제 cron 동작에 필수. 신청 안 하면 정기배송 결제 전부 실패.
- **웹훅 URL 등록**: `https://www.farmerstail.kr/api/payments/webhook`
- **billingAuth 성공/실패 URL**:
  - 성공: `https://www.farmerstail.kr/subscribe/billing-success`
  - 실패: `https://www.farmerstail.kr/subscribe/billing-fail`
- **환불/취소 권한 활성**

## 3. 전환 후 검증
- `https://www.farmerstail.kr/api/health` → `integrations.toss: true` 유지 확인
- 소액 실결제 1건 테스트 → Toss 콘솔에 거래 뜨는지 + 주문이 앱에 생기는지
- 환불 1건 테스트 → 부분/전체 환불 정상 처리되는지
- 정기배송 1건 빌링 등록 → 다음날 04:00(KST) subscription-charge cron 결제 확인
