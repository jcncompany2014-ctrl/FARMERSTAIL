## 확정 구조 (사장님 2026-07-24)
- 메뉴 28개 → **17개 · 4그룹**. 분석 4종은 삭제 대신 **"추후 개발" 그룹**으로 강등+쉬운말 리뉴얼.
- 탭 통합 6건 전부 진행. 페이지 병합 없이 **상단 탭바로 묶고 메뉴는 대표 1개만** (저위험).
- 기능 점검은 코드 배선 검증 + 프리뷰 실동작(사장님이 로그인 1회 협조).

```

              · 레시피 승인 · 고객[회원|답장|검색] · 환불 관리
💰 돈          매출·결제[리포트|원장] · 멤버십·스탬프
🛠️ 가끔        제품 관리 · 콘텐츠[블로그|FAQ|산지] · 알림[보내기|통계] · 설정[자동화|자동작업|알고리즘|발명플래그]
🔬 추후 개발    베타 테스트 · 가입 여정 분석 · 가입 시기별 분석 · 맞춤 분석
```
